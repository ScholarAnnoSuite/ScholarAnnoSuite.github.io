images.csv

# Name Definition Requirement
1 M_image_id The unique IDs of images in the dataset. | Img_index (default) Must have
2 M_image_name The name of the image. Must have
3 M_image_url The url link of the image. Must have
4 M_dispaly_order The order to show all images. | sort all images based on their appearance order at the dataset (default). Must have
5 O_caption_url The url link to the associated caption information of an image. Optional
6 O_paper_url The url link to the paper that contains the image. Optional
7 O_year The year when the paper was published. (used for the query panel). Optional

label_schema.csv

# Name Definition Requirement
1 M_label_id The unique IDs of labels used for the annotation task. | lab_index (default) Must have
2 M_label_name The name of the label. The label names should be unique under each category. Must have
3 M_label_type The type of the label (1: checkbox, 2: radio, 3: free text, 4: rectangle boundingbox, 0: category label, -1: inactive label) Must have
4 M_label_parent The parent of the current label, used for the label hierarchy. (root | label_id).
The root category is not the real label but an indicator to donote a group of labels, which is similar to the name of a dropdown menu. 		Must have
5 M_label_abbr The abbreviation of the label. (used for icon generation and results export) Must have
6 O_label_required Only used for root label. If the system need to check whether users annotated any labels under this category. Value: 1 | 0 (default) Optional
7 O_label_placeholder Only used for free-text labels. The placeholder for this label. | Empty string (default) Optional
8 O_label_icon The icon url of the label. | Empty string (default) Optional
9 O_label_bbox If this label will be used for localization tasks (bounding box annotation). Value: 1 | 0 (default) Optional
10 O_label_color The color used for the bbox button. (If the O_label_color is not provided, the system will generate colors automatically followed the methods in [1] [2]) Optional
